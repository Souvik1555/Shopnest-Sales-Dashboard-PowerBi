# 📊 ShopNest Store Sales & Delivery Performance Dashboard

## 🔹 Project Overview

This project analyzes sales performance, delivery efficiency, and customer behavior using Power BI.

## 🔹 Key Metrics

* Total Revenue: 13.59M
* Total Orders: 99K
* Delayed Orders: 44K
* Average Rating: 4

## 🔹 Key Insights

* Identified top-performing product categories
* Analyzed payment methods and sales trends
* Evaluated delivery delays and performance
* Analyzed customer ratings

## 🔹 Tools Used

* Power BI
* Excel
